# Mystery Movies

A romantic riddle game where the player discovers hidden letters through film and game-based enigmas, ultimately revealing a surprise travel destination.

## How to Play

1. Answer the entry question to begin
2. Solve 7 riddles based on movies and games
3. Each correct answer reveals a key letter
4. Unscramble the collected letters for the final answer
